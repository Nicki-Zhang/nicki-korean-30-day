- 第4课在 390×844、768px、1440px 三种 CSS 视口下完成真实 DOM 布局测试。
- 三种视口均满足 `document.scrollWidth === viewport width`，未检测到横向溢出元素。
- 标题保持正常横向换行，课程卡片、播放器和选项占据可用宽度。
- 所有按钮触控区域不小于 44px。
- 第1～3课代表性挑战反馈在简体中文、English、Tiếng Việt、日本語下完成真实 DOM 测试。
- 第4课听辨题在四种界面语言下完成作答前隐私与作答后反馈检查。
- 作答后显示音频原文、适用时的翻译、正确答案及简短教学解释。
- 播放完成后按钮只显示通用“再次播放”文案，不拼接韩语答案。
- 音频失败和设备韩语语音 fallback 不会将文字答案显示在播放器区域。
