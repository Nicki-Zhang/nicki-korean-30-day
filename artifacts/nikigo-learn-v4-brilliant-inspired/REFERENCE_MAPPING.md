## Adopted principles

| Reference principle | Nikigo V4 translation |
| --- | --- |
| Horizontal subject switch | Horizontal Stage tabs for K0 and K1 |
| Short subject introduction | Stage name, one learning goal, progress, original Stage illustration |
| Guided sequence | Vertical Module path with a light connector |
| Distinct unit imagery | One original Nikigo illustration per Module |
| Low-density scan | One Module per path step, Lessons move to Module detail |
| Strong current position | Current Module uses a violet border, light lavender surface, Korean anchor, and one primary action |

## Explicitly not copied

- Brilliant branding, blue system, icons, illustration assets, navigation, dimensions, shadows, and course content.
- Brilliant access rules. Nikigo connector lines express order only and never lock available courses.
- Brilliant subject taxonomy. Nikigo keeps the approved K0/K1 Stage and six explicit chapter IDs.
